# AI Recycling Frontend

## 프로젝트 소개

AI 기반 재활용품 분리배출 서비스의 프론트엔드입니다.

사용자가 재활용품 이미지를 업로드하면
AI 분석 결과를 바탕으로 올바른 분리배출 방법을 안내하는 앱 서비스입니다.

---

## 담당 역할

Frontend 100%

- 로그인 / 회원가입
- 이미지 업로드
- AI API 연동
- 결과 페이지
- 상세 안내 페이지
- 피드백 기능
